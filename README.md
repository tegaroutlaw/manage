<h2 align="center">
   ⇝ Install Locally Or On A VPS ⇜
</h2>

```console
git clone https://github.com/hitokizzy/Sri-Asih
cd Sri-Asih
pip3 install -U -r requirements.txt
cp sample_config.env config.env
```

<h3 align="center">
    Edit <b>config.env</b> with your own values
</h3>

<h2 align="center">
   ⇝ Run Directly ⇜
</h2>

```console
python3 -m Sriasih
```
